## how to use 
Draw a digit on the canvas.

Press Predict → it adds it to the equation.

Add operators (+, -, *, /) using the buttons.

Repeat steps for the next digit.

Press = → get the result!


## how to run the script 
```bash
python3 train.py
python3 handcalc.py
```
